2. gCap靶向捕获检测技术介绍

gCap利用探针杂交的方式捕获目标区域，以检测目标位点相关的遗传变异信息。与全基因组重测序相比，gCap显著提升目标位点的测序深度，使目标位点的测序深度可达100×以上，确保了基因分型的准确性。

其工作原理为：采用“平铺式”+“叠瓦式”相结合的探针设计方案，对目标区域进行多层探针设计，确保探针覆盖目标检测区域，并使用生物素标记的NTP和核酸类似物作为原料进行探针合成。特异性探针能够与目标区域基因组文库进行液相杂交，杂交完成后使用链霉亲和素磁珠来捕获探针及目标区域文库的杂交产物。将捕获得到的目标区域文库富集后，使用二代测序平台进行高通量测序，可检测目标区域的 SNP、InDel、CNV、SV等变异。


3. 数据质控与捕获效果统计


3.1 参考基因组比对

本模块统计各样本原始测序数据经质量过滤后，比对至参考基因组的结果。该结果是数据质控的第一步，用于评估测序数据的可利用比例。高比对率是后续准确基因分型的前提。此模块详情可查看结果文件stat/bwa_result.xls

  • Mapping_rate(%)：参考基因组比对率，计算公式为：Mapped_reads / Clean_reads × 100%。


3.2 捕获效果统计

本模块汇总了gCap靶向捕获测序的关键技术指标，包括位点检出率、捕获率、覆盖度及平均深度。用于系统性评估本次实验的捕获效果。此模块详情可查看结果文件stat/stat.xls

{% for sample in buhuo_data %} {% endfor %}
Sample Site_detection_rate(%) Capture_rate(%) Coverage(%) Average_depth
{{ sample.Sample|string }} {{ "{:.2f}".format(sample.site_detection_rate) }} {{ "{:.2f}".format(sample.capture_rate) }} {{ "{:.2f}".format(sample.coverage) }} {{ "{:,.2f}".format(sample.average_depth) }}

表头说明:

  • Sample：样本名称;
  • Site_detection_rate(%)：位点检出率，计算公式为：被检测到的目标位点数 / 芯片位点数 × 100%;
  • Capture_rate(%)：捕获率，计算公式为：位于探针捕获区域的reads数 / 过滤后的reads总数 × 100%;
  • Coverage(%)：覆盖度，探针捕获区域中被测序reads覆盖的区域比例;
  • Average_depth：平均深度，探针捕获区域的平均测序深度。


4. 目标位点分析


4.1 目标位点基因型分型

本模块对每个样本的所有目标位点进行基因型分型。为保证分型结果准确，当位点覆盖深度≥4×时，输出该位点基因型结果；反之按缺失处理。此外，本批次中所有多等位基因位点均按缺失基因型处理。本模块将交付plink格式的结果文件。


4.2 各样本目标位点基因型统计

本模块从样本维度进行统计，展示了每个样本在目标位点上的基因型分布概况，包括缺失位点、纯合参考型、纯合变异型及杂合型的比例。该结果有助于评估每个样本的基因分型质量与数据完整性，可用于识别可能存在DNA质量或污染问题的异常样本。此模块详情可查看结果文件SNP/chip_sample_stat.xls
